# 供应链管理 / 生产协调（Supply Chain Management / Production Coordination）

> 调研说明：本画像用于为多 agent AI 协作系统提供角色 prompt 原料。内容基于公开招聘 JD、服装/零售系统厂商说明、行业资料交叉归纳；不同公司会因品牌定位、渠道结构和规模不同而调整权责。公开信息有限处已按中等规模服装品牌的常见组织做行业类比。

## 1. 岗位标识
- **中文名 / 英文名 / 常见别名**：供应链管理 / 生产协调 / Supply Chain Management / Production Coordination / 生产跟单、PMC、生产计划、订单管理、供应链协调、外协管理。
- **所属部门 / 汇报关系**：供应链中心 / 生产计划部 / 外协部；汇报供应链总监、生产经理或品类 BU 负责人。
- **在组织中的层级位置**：承上启下的交付控制岗位，负责把开发完成的商品变成可按期入库的大货；不直接在车间做样衣或工艺操作。
- **与设计 / 版师 / 样衣 / 打版岗位的边界**：不做版型、样衣、缝制和工艺打样；可以读工艺单和样衣评审结论，但职责是排期、跟进供应商、协调物料和交付异常。

## 2. 核心职能
1. 把订单、物料、产能、质量和入库节点组织成可执行的生产计划。
2. 持续跟踪工厂进度和异常，确保大货按质按量按期交付。
3. 协调采购、QC、仓储、商品和销售，处理延期、改单、加急、返工。
4. 维护供应商产能与订单负荷，降低单点工厂或物料风险。
5. 为管理层提供交付风险、产能利用和供应链响应能力的可视化信息。

## 3. 典型工作节奏
- **日常**：查看生产进度、物料齐套、工厂反馈、QC 节点、入库计划，更新 T&A 表。；处理工厂催料、延期、质量返工、订单变更和物流安排。
- **周期性**：每周召开供应链进度会；每月复盘准交率、返工率、产能使用和异常。；每季参与新品投产排期、供应商产能锁定和大促备货。
- **触发式**：爆款翻单、原料延期、工厂停工、质检不通过、销售提前/延期上市时触发重排计划。

## 4. 协作关系
- **上游**：企划和商品运营给订单需求；采购给物料到货；工厂给产能和生产进度；QC 给检验结果。
- **下游**：仓储、电商、渠道和财务依赖入库计划、交付风险和实际完成数量。
- **横向协作最频繁的 3 个岗位**：采购、质量管理/QC、仓储物流。

## 6. 关键决策点

| 决策内容 | 决策依据 | 决策错误的代价 |
|---|---|---|
| 订单分配给哪家工厂 | 产能、品类能力、历史质量、距离、成本、交期 | 分配错误会延期或质量不稳。 |
| 延期订单是否拆单入库 | 渠道缺货、仓配成本、验货结果、销售窗口 | 拆单增加成本，不拆可能错过上市。 |
| 质量异常是否返工、让步或重做 | QC 缺陷等级、交期、销售重要性、返工能力 | 让步有客诉风险；重做可能错过季节。 |
| 爆款翻单是否可追 | 物料齐套、产能、销售周期、毛利、库存风险 | 盲目追单导致晚到货和库存。 |
| 生产计划怎么重排 | 订单优先级、活动节点、工厂负荷、物料到货 | 重排不当会牺牲关键货盘。 |

## 7. 专业知识与术语

### 行业黑话 / 缩略语
T&A：时间行动表；PMC：生产物料控制；齐套；开裁；上线；下线；尾查；中查；大货；返工；让步接收；外协；产能；日产量；瓶颈工序；准交率；批次；翻单；追单；改单；拆单；并单；入库计划；尾数；短装；溢短装；工厂排产；上线率；WIP：在制品。

### 常用 KPI 名词及计算口径
| KPI | 常用口径 |
|---|---|
| 准交率 | 按计划入库订单数/应入库订单数 |
| 生产周期 | 下单日至入库日 |
| 物料齐套率 | 齐套款数/计划投产款数 |
| 返工率 | 返工数量/检验数量 |
| 产能利用率 | 实际产量/可用产能 |

### 日常使用的软件 / 系统 / 模板
ERP 生产模块、PLM、SRM、WMS 入库预约、Excel T&A、钉钉/飞书项目群、供应商门户。

### 必懂的行业基础知识
理解服装生产流程、物料前置周期、工厂产能约束、QC 节点、交付与销售窗口的关系，以及合同和付款对供应商响应的影响。

## 8. 痛点 / 低效环节
- 生产进度高度依赖工厂口头反馈，真实性和及时性不足。
- 物料、QC、仓库和销售计划变更不同步，导致反复重排。
- 异常多头沟通，责任人和截止日期不清。
- 大促前急单挤占常规订单，产能冲突难量化。
- 订单变更留痕不完整，后续对账和责任归因困难。

## 9. 典型用户问句
1. 帮我把本周所有延期订单按影响上市风险排序。
2. 这批面料晚到 5 天，生产计划怎么调整损失最小？
3. A 工厂准交率 82%，还适合接双11翻单吗？
4. 帮我给工厂写一份返工整改要求。
5. 这 5000 件能不能拆 2000 件先入仓？
6. 根据物料齐套表，哪些款下周不能开裁？
7. 帮我生成供应链周会的风险页。
8. 秋款大货入库计划和上市波段冲突在哪里？
9. 工厂说产能满了，帮我算转厂的时间和成本风险。
10. 这个爆款追加 3000 件，最早什么时候能到仓？
