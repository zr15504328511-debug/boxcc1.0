## 1. 岗位标识
- **中文名 / 英文名 / 常见别名**：数据分析 / BI / Data Analyst / Business Intelligence / BI 分析师、经营分析、数据中台分析、商业分析、数据产品。
- **所属部门 / 汇报关系**：数据部 / IT 数据中台 / 经营分析部；汇报数据负责人、CIO、经营分析负责人或 CEO 办公室；服务商品、渠道、电商、供应链和财务。
- **在组织中的层级位置**：跨部门数据中台岗位，负责指标口径、数据看板、专题分析和数据治理；与商品分析不同，BI 更强调全公司指标体系和数据基础设施。
- **与设计 / 版师 / 样衣 / 打版岗位的边界**：不做服装设计、不做业务最终拍板；负责数据口径、洞察和工具，业务部门决定行动。

## 2. 核心职能
1. 把分散在 ERP、POS、OMS、WMS、电商平台和财务中的数据统一成可用指标。
2. 设计和维护销售、库存、会员、渠道、供应链、财务等经营看板。
3. 通过专题分析定位业务问题，如库存积压、转化下降、退货异常或毛利波动。
4. 建立指标口径和数据质量规则，减少部门之间“各算各的”。
5. 为业务预测、预算、补货、活动和管理会议提供数据支持。

## 3. 典型工作节奏
- **日常**：监控数据同步、看板异常和业务临时取数；回应口径问题。；检查关键指标波动并提醒相关部门。
- **周期性**：每周经营周报；每月管理层经营分析、指标口径校准和数据质量复盘。；每季支持预算、订货、商品复盘和供应链绩效分析。
- **触发式**：GMV/库存/退货/毛利异常、系统上线、数据口径争议、大促复盘时触发专题分析。

## 4. 协作关系
- **上游**：IT 提供数据链路；业务部门提供规则和解释；财务给成本和结算口径；平台后台给渠道数据。
- **下游**：管理层、商品、渠道、电商、供应链、财务使用 BI 看板和分析报告。
- **横向协作最频繁的 3 个岗位**：IT/数据中台、财务预算、商品运营。

## 6. 关键决策点

| 决策内容 | 决策依据 | 决策错误的代价 |
|---|---|---|
| 哪个指标波动值得预警 | 历史波动、业务日历、异常阈值、数据质量 | 误报太多降低信任，漏报错过风险。 |
| 采用哪个口径做管理报表 | 业务定义、财务一致性、系统可得性、历史可比 | 口径错会导致部门争议和错误决策。 |
| 专题分析从哪些维度切 | 品类、渠道、时间、用户、价格、库存、供应商 | 维度错会找不到真实原因。 |
| 看板刷新频率和权限怎么设 | 数据敏感度、业务时效、系统成本、使用人群 | 权限过宽有风险，刷新慢影响决策。 |
| 模型建议是否可给业务使用 | 数据量、稳定性、解释性、业务约束 | 模型误导会造成错误补货或预算。 |

## 7. 专业知识与术语

### 行业黑话 / 缩略语
BI；ETL；数据仓库；数据集市；指标口径；维度；事实表；主数据；数据血缘；同比；环比；漏斗；留存； cohort；GMV；UV；PV；CTR；CVR；客单价；毛利；库存周转；动销；售罄；缺失值；异常值；权限；脱敏；API；数据质量；仪表盘；预警。

### 常用 KPI 名词及计算口径
| KPI | 常用口径 |
|---|---|
| 报表准时率 | 按时刷新报表数/应刷新报表数 |
| 数据质量问题关闭率 | 已关闭问题/问题总数 |
| 看板使用率 | 活跃访问用户或次数/目标用户 |
| 分析采纳率 | 被业务采纳建议/提出建议 |
| 取数响应时效 | 需求提交到交付时间 |

### 日常使用的软件 / 系统 / 模板
SQL、Python/R、Power BI/Tableau/FineBI、Excel、数据仓库、ERP/POS/OMS/WMS/CRM、埋点平台、权限系统。

### 必懂的行业基础知识
懂服装款色码、渠道和库存基础，能把业务语言映射为指标；同时理解数据治理、口径管理和隐私合规。

## 8. 痛点 / 低效环节
- 源系统多且字段不标准，商品主数据不干净。
- 业务临时取数多，分析师被动救火。
- 管理层和部门口径不一致，会议上先争数据再谈问题。
- 平台数据无法稳定自动接入，仍需手动导出。
- 业务希望“给结论”，但因果数据不足。

## 9. 典型用户问句
1. 帮我查一下上周毛利率下降主要由哪些品类贡献。
2. 把动销率、售罄率、库销比的口径写成统一指标字典。
3. 这个看板 GMV 和财务收入差 8%，可能差在哪些口径？
4. 帮我做一个商品库存预警规则。
5. 双11复盘需要哪些数据表和维度？
6. 这 30 家门店客流下降但转化提升，怎么解释？
7. 帮我把退货率按尺码、面料、渠道拆一下。
8. 做新品销量预测需要哪些历史数据？
9. 这次数据异常是源系统延迟还是业务真的波动？
10. 把管理层经营周报自动化前，字段清单怎么列？
