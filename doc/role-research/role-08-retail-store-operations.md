## 1. 岗位标识
- **中文名 / 英文名 / 常见别名**：零售门店运营 / 督导 / Retail Store Operations / Area Supervisor / 零售督导、区域督导、门店运营、店铺管理、营运督导。
- **所属部门 / 汇报关系**：零售运营部 / 直营事业部 / 区域运营；汇报零售经理、区域经理或销售总监；对店长和导购有业务指导关系。
- **在组织中的层级位置**：终端执行管理岗位，负责门店标准、业绩、人员、陈列、服务和现场问题闭环。
- **与设计 / 版师 / 样衣 / 打版岗位的边界**：不做设计、不打版、不制作陈列道具；可反馈穿搭和顾客意见，但不替代视觉陈列师或设计开发岗位。

## 2. 核心职能
1. 把总部策略落到门店现场，确保销售、服务、陈列、库存和人员标准被执行。
2. 通过巡店、培训、数据复盘和辅导提升店效、人效和顾客体验。
3. 把门店一线问题反馈给商品、营销、培训和客服，形成改进。
4. 协同商品运营做补调退和陈列主推，提升门店售罄与连带。
5. 维护门店日常运营秩序，降低损耗、客诉和执行偏差。

## 3. 典型工作节奏
- **日常**：看门店销售、客流、转化、库存、排班和客诉；处理门店突发问题。；在群内跟进销售进度、主推执行、活动落地和人员状态。
- **周期性**：每周巡店和店长会议；每月店铺经营复盘、培训和考核。；每季参与新品培训、陈列调整、订货会反馈和门店评级。
- **触发式**：新店开业、业绩下滑、客诉、人员离职、盘点差异、活动执行差时触发专项辅导。

## 4. 协作关系
- **上游**：销售目标来自零售/销售负责人；商品与营销给主推、活动和陈列指引；HR/培训给人员和课程。
- **下游**：店长、导购、商品运营、营销、HR 使用督导的巡店报告、培训反馈和门店数据。
- **横向协作最频繁的 3 个岗位**：商品运营、销售渠道运营、HR/培训。

## 6. 关键决策点

| 决策内容 | 决策依据 | 决策错误的代价 |
|---|---|---|
| 业绩下滑优先查什么 | 客流、转化、客单、连带、货品、人员、竞品 | 归因错会让改进动作无效。 |
| 主推款在门店怎么执行 | 库存、陈列位置、顾客画像、导购话术、活动资源 | 执行差会浪费货盘和营销资源。 |
| 门店是否需要调货/退货 | 销售速度、断码、库销比、陈列容量、区域需求 | 调错货会损失销售或加大库存。 |
| 哪个导购需要培训或激励 | 销售、人效、连带、服务评价、店长反馈 | 人员管理不当影响团队稳定和业绩。 |
| 客诉现场如何处理 | 公司政策、顾客情绪、证据、金额、品牌影响 | 处理过轻升级舆情，过度补偿增加成本和规则破坏。 |

## 8. 痛点 / 低效环节
- 巡店记录大量图片和口头反馈，难形成可追踪整改闭环。
- 门店销售下滑常被简单归因为人，缺少商品/客流/活动拆解。
- 新品卖点培训到导购时被压缩或走样。
- 门店调拨和退仓审批慢，错过销售机会。
- 一线顾客反馈无法及时回到商品企划和内容团队。

## 9. 典型用户问句
1. 帮我分析这家店本周业绩下滑是客流还是转化问题。
2. 把今天巡店照片整理成整改清单。
3. 这款主推连衣裙导购卖不动，帮我写 5 句试穿话术。
4. 门店库存偏深，哪些款建议退仓？
5. 新店开业前还缺哪些物料和培训动作？
6. 把本月店效 TOP/TAIL 店铺做一页复盘。
7. 客人投诉羊毛衫起球，门店怎么回复更稳妥？
8. 帮我给店长会写一份下周重点动作。
9. 这个区域导购离职率高，可能原因有哪些？
10. 新品培训后做一个 10 题测验。
