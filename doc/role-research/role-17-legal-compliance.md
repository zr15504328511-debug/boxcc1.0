## 1. 岗位标识
- **中文名 / 英文名 / 常见别名**：法务 / 合规 / Legal / Compliance / 合同法务、商标法务、广告法合规、产品合规、知识产权、内控合规。
- **所属部门 / 汇报关系**：法务部 / 合规部 / 总经办；汇报法务负责人、CFO、CEO 或集团法务。
- **在组织中的层级位置**：风险控制岗位，覆盖合同、知识产权、广告宣传、消费者权益、产品标签、劳动和数据合规。
- **与设计 / 版师 / 样衣 / 打版岗位的边界**：不做设计、不决定商品上市；法务提供风险判断、条款和合规边界，业务在风险知情后做商业决策。

## 2. 核心职能
1. 审查采购、供应商、渠道、达人、联名、服务和劳动等合同，控制权利义务和违约风险。
2. 管理商标、著作权、图案/素材授权和侵权维权，保护品牌资产。
3. 审核广告文案、直播话术、详情页和公关口径，降低虚假宣传、广告法和平台违规风险。
4. 跟进服装标签、成分、安全标准、平台规则和跨境/地区合规要求。
5. 处理纠纷、投诉、律师函、行政检查和舆情法律风险。

## 3. 典型工作节奏
- **日常**：审合同、看宣传文案、答业务咨询、处理投诉和证据留存。；跟进商标、授权、用印和合同归档。
- **周期性**：每周合同台账和风险事项；每月合规培训、纠纷复盘和制度更新。；每季检查重点平台页面、直播话术和供应商合同模板。
- **触发式**：联名合作、广告投放、达人合作、质量争议、消费者投诉、侵权线索、行政抽查时触发法务介入。

## 4. 协作关系
- **上游**：业务部门提交合同、文案、活动规则和证据；QC/采购提供检测与供应商资料；品牌/内容给传播口径。
- **下游**：采购、渠道、电商、PR、内容、HR、管理层使用法务意见、合同模板、风险提示和争议处理方案。
- **横向协作最频繁的 3 个岗位**：品牌营销、PR、采购。

## 6. 关键决策点

| 决策内容 | 决策依据 | 决策错误的代价 |
|---|---|---|
| 合同条款是否可接受 | 金额、责任、违约、付款、交付、知识产权、争议解决 | 条款失控会带来赔偿、履约或 IP 风险。 |
| 宣传表达是否合规 | 事实依据、检测报告、广告法、平台规则、消费者理解 | 违规会被处罚、下架或投诉。 |
| 是否发起维权 | 证据强度、权利稳定性、成本、商业影响 | 维权不足损害品牌，过度维权浪费资源。 |
| 消费者争议怎么处理 | 证据、平台规则、法律责任、舆情影响、成本 | 处理不当导致仲裁、诉讼或负面传播。 |
| 新品标签是否可上市 | 成分、标准、检测、吊牌、洗护、进口/跨境规则 | 标签错误可能召回、处罚或下架。 |

## 7. 专业知识与术语

### 必懂的行业基础知识
懂合同、知识产权、广告宣传、消费者权益、产品标签和纺织安全标准基础；理解电商直播和达人合作的实际风险。

## 8. 痛点 / 低效环节
- 业务常在上线前最后一刻才提交审核，修改空间小。
- 素材版权和授权范围记录不完整。
- 达人/MCN 合同履约证据分散，争议时难取证。
- 服装成分、标签、详情页和直播话术不一致。
- 合规意见容易被认为“阻碍业务”，需要给可替代方案。
