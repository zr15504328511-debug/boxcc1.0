## 1. 岗位标识
- **中文名 / 英文名 / 常见别名**：仓储物流 / Warehouse and Logistics Operations / 仓库运营、WMS 专员、物流专员、履约、配货、库存管理员。
- **所属部门 / 汇报关系**：仓储物流部 / 供应链履约中心；汇报仓储经理、物流经理或供应链总监。
- **在组织中的层级位置**：后端履约岗位，管理从收货、上架、库存、拣货、复核、发货、退货到盘点的现场与系统流程。
- **与设计 / 版师 / 样衣 / 打版岗位的边界**：不决定款式设计、不做商品企划；对商品只按款色码和库存状态管理，不修改实物设计和质量标准。

## 2. 核心职能
1. 让商品库存从“账上有”变成“找得到、拣得准、发得快、可追溯”。
2. 管理入库、上架、拣配、复核、发运、退货和盘点流程，降低错发漏发。
3. 通过仓位、波次、批次、条码和 WMS 规则提升人效和库存准确率。
4. 与商品运营和渠道协同，保障大促、门店补货、调拨和退货处理。
5. 沉淀物流成本、时效和异常数据，为渠道履约策略提供依据。

## 3. 典型工作节奏
- **日常**：收货验数、上架、拣货、复核、打包、发运、异常件处理、库存调整。；更新库存差异、发货日报和承运商异常。
- **周期性**：每周循环盘点和库位优化；每月复盘库存准确率、发货时效、物流成本。；大促前做库存冻结、人员排班、波次策略和包材准备。
- **触发式**：爆单、退货高峰、盘点差异、错发投诉、门店紧急补货、承运商延误时触发应急处理。

## 4. 协作关系
- **上游**：采购/生产协调给入库计划；QC 给放行状态；商品运营给补调退和活动货盘；电商/渠道给订单。
- **下游**：门店、电商平台、客服、财务和商品运营依赖发货、库存、退货和物流异常结果。
- **横向协作最频繁的 3 个岗位**：商品运营、电商运营、客服/VOC。

## 6. 关键决策点

| 决策内容 | 决策依据 | 决策错误的代价 |
|---|---|---|
| 订单按什么波次拣货 | 订单截止时间、SKU 集中度、承运商、活动优先级 | 波次错误会降低人效或延误发货。 |
| 库存差异是否调整账务 | 盘点证据、出入库记录、责任归因、财务规则 | 随意调账会掩盖损耗；不调会影响可售。 |
| 退货商品是否可二次销售 | 污损程度、吊牌包装、质量标准、平台规则 | 误判会引发二次客诉或损失可售库存。 |
| 门店调拨是否走大仓或直发 | 距离、库存深度、时效、成本、承运商能力 | 路线错误会增加时效和物流成本。 |
| 大促是否加人加班 | 订单预测、历史人效、截单承诺、包材设备 | 不足会爆仓，过量会成本浪费。 |

## 7. 专业知识与术语

### 行业黑话 / 缩略语
WMS：仓储管理系统；OMS：订单管理系统；PDA；条码；RFID；收货；上架；拣货；复核；打包；发运；波次；库位；库区；先进先出；可售库存；冻结库存；预占库存；残次品；二销；循环盘点；账实差异；错发；漏发；截单时间；妥投；逆向物流；承运商；时效达成率。

### 常用 KPI 名词及计算口径
| KPI | 常用口径 |
|---|---|
| 库存准确率 | 账实一致 SKU 数/盘点 SKU 数 |
| 发货及时率 | 按承诺时间发出订单/应发订单 |
| 错发漏发率 | 错漏订单/总订单 |
| 人效 | 件数或订单数/人/小时 |
| 退货入库及时率 | 按时完成退货质检入库数/退货到仓数 |

### 日常使用的软件 / 系统 / 模板
WMS、OMS、ERP、TMS、PDA、电子面单、快递后台、BI 看板、Excel 排班和盘点模板。

### 必懂的行业基础知识
懂款色码 SKU、多渠道库存、批次与库位、正逆向流程、承运商时效、包材、盘点和库存财务影响。

## 8. 痛点 / 低效环节
- 服装款色码多，错拣和混放风险高。
- 大促订单波峰极大，仓库人力和包材准备容易不足。
- 退货原因和质检结果不规范，影响二次销售和 VOC 分析。
- OMS/WMS/ERP 库存同步延迟导致超卖或缺货。
- 盘点差异归因困难，现场记录不完整。

4. 库存准确率下降到 97%，先查哪些环节？
5. 帮我写一个双11仓库临时工上岗 SOP。
6. 这 300 件门店急补，走顺丰还是调拨车？
7. 把本周错发漏发案例整理成培训材料。
8. 这批到仓货还没 QC 放行，能不能先上架冻结？
9. 帮我生成今日发货日报给电商运营。
10. 退货入库慢影响退款率，流程哪里最可能堵？
