# 服装检测报告读法

检测报告不是只看“合格/不合格”。采购、QC、商品和客服都应能读懂报告中的样品信息、检测依据、项目结果、判定规则和限制条件。错误引用报告会导致错误放行、误导宣传或售后争议。

## 一、先看样品是否匹配

报告首页通常包含委托单位、样品名称、颜色、款号或面料编号、成分、送检日期和检测机构。读取时先确认报告对应的是大货、产前样、面料样还是竞品样。若报告只覆盖黑色面料，不应直接证明同款浅色或拼色款全部合格；若报告是面料检测，不一定覆盖成衣缝制、粘衬、印花和辅料后的表现。

## 二、再看检测依据

同一项目可能有不同测试标准和条件。缩水率要看水洗、干洗还是手洗模拟；色牢度要看皂洗、摩擦、汗渍、耐光还是升华；起球要看使用圆轨迹、马丁代尔还是箱式。报告中的条件决定了结果能被怎样引用。

| 项目 | 必看字段 | 常见误读 |
| --- | --- | --- |
| 纤维含量 | 成分比例、允差、测试方法 | 把面料样比例当成整衣所有部位 |
| 尺寸变化率 | 洗涤程序、干燥方式、经纬向 | 忽略吊牌洗护方式 |
| 色牢度 | 干摩/湿摩、沾色对象 | 只看最高等级，不看湿摩 |
| 起球 | 方法、次数、评级照片 | 把 4 级解释为不会起球 |
| 甲醛/pH/异味 | 标准限值、产品类别 | 忽略婴童或贴身类别差异 |

## 三、如何用于业务决策

开发放行：关注是否达到品牌内控线，而不是只达到国家最低线。高客单价、贴身、浅色搭配、强营销卖点的产品应提高要求。

内容宣传：只能引用报告能证明的事实。可以写“经检测，纤维含量为绵羊毛 80%”，不能因为报告合格就写“顶级羊毛”“永不变形”。涉及 UPF、防水、抗菌、环保认证时，必须确认检测项目、标准和授权范围。

售后处理：报告合格可以作为判断依据之一，但不能替代订单事实。客服仍需收集穿着时长、洗涤方式、局部照片、整体照片和洗标照片。若消费者问题与报告未覆盖项目有关，应转 QC 或供应链复核，避免直接用无关报告拒绝。

## 四、归档要求

报告文件名建议包含款号、颜色、面料编号、检测项目和日期。大货上市前，采购负责上传供应商报告，QC 负责标记是否采用第三方复测，商品和内容团队只使用已归档且状态为有效的报告。
