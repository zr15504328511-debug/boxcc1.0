## 1. 岗位标识
- **中文名 / 英文名 / 常见别名**：电商运营（天猫 / 抖音 / 小红书 / 直播） / E-commerce Operations / 天猫运营、抖音运营、直播运营、小红书店铺运营、平台运营、店长。
- **所属部门 / 汇报关系**：电商事业部 / 新零售部 / 内容电商部；汇报电商总监、平台负责人或品牌 GM。
- **在组织中的层级位置**：线上渠道经营岗位，通常按平台、店铺或直播间分工；需协调商品、内容、投放、客服、仓储和财务。
- **与设计 / 版师 / 样衣 / 打版岗位的边界**：不设计服装实物，不拍摄制作所有素材；负责平台生意、活动、货盘、页面、流量和转化，对内容/视觉提出需求。

## 2. 核心职能
1. 对平台 GMV、利润、流量、转化和店铺健康负责。
2. 把商品货盘、平台活动、内容素材、投放和仓储履约组织成线上销售计划。
3. 实时监控店铺/直播数据，调整价格、流量、话术、库存和页面。
4. 理解不同平台规则和用户心智，制定差异化运营打法。
5. 把线上用户反馈和退货原因反哺商品、内容和客服。

## 3. 典型工作节奏
- **日常**：看 GMV、访客、转化、客单、广告消耗、库存、客服和售后指标。；更新活动价格、商品上下架、直播排品、投放素材和店铺页面需求。
- **周期性**：每周平台经营复盘；每月活动排期、预算和货盘计划。；618/双11/年货节等大促前做目标、货盘、投放、直播和仓配预案。
- **触发式**：流量骤降、转化异常、平台规则变化、直播爆单、差评/退货激增、竞品大促时触发应急调整。

## 4. 协作关系
- **上游**：商品运营给货盘和库存；营销/内容给素材；平台小二/达人给活动资源；仓储和客服给履约反馈。
- **下游**：向商品、仓储、客服、财务、内容、投放提供活动计划、销售预测、页面需求和复盘。
- **横向协作最频繁的 3 个岗位**：内容/社媒、客服/VOC、仓储物流。

## 6. 关键决策点

| 决策内容 | 决策依据 | 决策错误的代价 |
|---|---|---|
| 大促货盘选哪些 SKU | 库存、毛利、历史转化、平台趋势、退货率、内容资源 | 选错货浪费流量或造成高退货。 |
| 广告预算怎么分配 | ROI、转化率、素材表现、库存、活动阶段 | 预算错配会造成亏损或错过爆发。 |
| 直播间是否改价格/机制 | 在线人数、点击、成交、库存、毛利、竞品 | 临场决策错误会损害利润或转化。 |
| 新品上架节奏怎么排 | 平台活动、库存入仓、内容素材、搜索热度 | 上架早无库存，晚了错过流量窗口。 |
| 差评/退货激增如何处理 | 退货原因、商品批次、客服记录、物流异常 | 处理慢会影响评分、流量和品牌口碑。 |

## 7. 专业知识与术语

### 行业黑话 / 缩略语
GMV；UV；PV；CTR：点击率；CVR：转化率；客单价；UV价值；ROI/ROAS；GPM：千次观看成交；停留时长；加购率；收藏率；退款率；DSR；坑位；商品卡；短视频种草；达人分销；直播间排品；场控；福袋；憋单；千川；直通车；超级推荐；搜索权重；自然流量；付费流量；人群包；投产比。

### 常用 KPI 名词及计算口径
| KPI | 常用口径 |
|---|---|
| GMV | 成交金额，通常含退款前口径需注明 |
| 转化率 | 支付买家数/访客数，直播可用成交人数/观看人数 |
| 客单价 | 支付金额/支付买家数 |
| ROI/ROAS | 成交金额/广告花费 |
| 退款率 | 退款金额或订单/支付金额或订单 |

### 日常使用的软件 / 系统 / 模板
天猫/淘宝后台、生意参谋、抖店罗盘、巨量千川、小红书商家后台、聚水潭/旺店通、ERP/OMS、客服系统、飞瓜/蝉妈妈等数据工具。

### 必懂的行业基础知识
懂平台规则、流量入口、活动机制、商品卡、直播转化漏斗、内容种草、库存冻结、广告投放和电商售后评分。

## 8. 痛点 / 低效环节
- 平台后台数据口径多，日报整合耗时。
- 活动价格和库存同步容易出错，引发超卖或亏损。
- 直播间临场决策高度依赖经验，复盘却缺少结构化记录。
- 内容、投放、商品、仓储节奏不一致。
- 退货和差评信息没有及时回到商品决策。
