# 面料 / 辅料 / 成品采购（Fabric, Trim and Finished Goods Sourcing / Procurement）

## 1. 岗位标识
- **中文名 / 英文名 / 常见别名**：面料 / 辅料 / 成品采购 / Fabric, Trim and Finished Goods Sourcing / Procurement / 面料采购、辅料采购、成衣采购、开发采购、供应商开发、Sourcing。
- **所属部门 / 汇报关系**：供应链中心 / 采购部 / 商品供应部；通常汇报采购经理、供应链总监或品类供应链负责人。
- **在组织中的层级位置**：供应链前段商务岗位，负责供应商、价格、交期、合同和物料订单；中型公司常按面料、辅料、成品或品类分工。
- **与设计 / 版师 / 样衣 / 打版岗位的边界**：不做面料原创设计、不打样衣、不改版；可参与选样和测试，但职责是找资源、谈成本、下采购单、保交期和质量。

## 2. 核心职能
1. 把商品开发和生产需求转成可采购的物料/成品订单，并确保成本、质量、交期可控。
2. 开发、评估、维护供应商，建立可替代供应资源，降低单一供应风险。
3. 通过询价、比价、核价和合同条款控制采购成本与付款风险。
4. 跟踪面辅料和成品交期，提前暴露断料、涨价、最低起订量和品质异常。
5. 把采购信息准确回写 ERP/PLM，保证成本核算、生产排期和财务付款有依据。

## 3. 典型工作节奏
- **日常**：处理询价、比价、下单、供应商交期确认、样品寄送、到货异常和付款资料。；与供应链、QC、财务核对采购订单、合同、发票、对账单。
- **周期性**：每周更新采购进度和供应商交付；每月做成本节省、供应商绩效和异常复盘。；每季参与开发前资源准备、战略供应商谈判和价格库更新。
- **触发式**：物料断供、价格波动、质量不合格、订单变更、急单翻单、供应商产能冲突时触发应急采购或替代方案。

## 4. 协作关系
- **上游**：商品企划和设计开发提出材质、成本和交期需求；供应链给生产计划；QC 给测试和验货标准。
- **下游**：向生产协调、财务、QC、仓储提供采购订单、交期、到货和供应商信息。
- **横向协作最频繁的 3 个岗位**：供应链生产协调、质量管理/QC、财务成本会计。

## 6. 关键决策点

| 决策内容 | 决策依据 | 决策错误的代价 |
|---|---|---|
| 选哪家供应商下单 | 价格、质量记录、产能、交期、账期、认证、历史合作 | 选错供应商会造成延期、返工或成本失控。 |
| 是否接受 MOQ 或改规格 | 销售预测、库存风险、成本影响、替代供应 | MOQ 过高会积压物料，改规格可能影响产品定位。 |
| 是否启用替代面辅料 | 测试结果、色差、手感、成本、交期、设计确认 | 替代不当会引发质量投诉或品牌感下降。 |
| 价格上涨是否转嫁或重谈 | 市场行情、历史价格、毛利红线、供应商关系 | 处理不当会损害毛利或供应稳定。 |
| 异常到货是否让步放行 | QC 结果、生产窗口、返工成本、销售节点 | 放行有质量风险，拒收有交期风险。 |

## 7. 专业知识与术语

### 行业黑话 / 缩略语
RFQ：询价；PO：采购订单；MOQ：最低起订量；MCQ：最低染色量；Lead Time：交期；LT：周期；账期；含税价；未税价；单耗；损耗率；克重；门幅；色牢度；缩水率；缸差；色差；封样；大货；客供料；替代料；供应商准交率；合格率；来料检验；对账；扣款；L/C；FOB；CMT；FOB 成衣；OEM；ODM。

|---|---|

### 日常使用的软件 / 系统 / 模板
SRM、ERP 采购模块、PLM 物料库、Excel 比价表、电子合同、邮件/钉钉审批、第三方检测报告平台。

### 必懂的行业基础知识
要懂面辅料基础、常见测试、起订量与染整周期、采购合同条款、税率和付款、物料编码、批次追溯。

## 8. 痛点 / 低效环节
- 供应商报价格式不统一，比价和历史价查询靠手工。
- 开发期需求变化频繁，采购单反复改。
- 样卡、封样、检测报告散落在线下和聊天记录中。
- 交期异常发现晚，生产窗口被压缩。
- 供应商绩效数据不完整，淘汰和培养难量化。

## 9. 典型用户问句
1. 帮我把这三家拉链供应商报价做一版含税/未税对比。
2. 这块面料 MOQ 3000 米，但订单只用 1800 米，怎么评估剩余风险？
3. 给供应商写一封交期延期的违约催告邮件，语气强硬但保留合作。
4. 这次色差报告 Delta E 偏高，能不能让步接收？
5. 帮我整理一份秋冬主推款的面辅料到货风险清单。
6. 同规格莱赛尔面料，A 厂便宜 8%，但准交率差，要不要切过去？
7. 把供应商月度绩效按价格、交期、质量打分。
8. 采购合同里关于延期交货的条款帮我检查一下。
9. 这批辅料短装 3%，对生产排期影响多大？
10. 帮我写一份替代扣具方案给设计确认。
