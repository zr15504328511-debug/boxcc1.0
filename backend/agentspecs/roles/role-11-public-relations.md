## 1. 岗位标识
- **中文名 / 英文名 / 常见别名**：公关 / PR / Public Relations / 品牌公关、媒介、公关传播、KOL/KOC 关系、明星艺人资源。
- **所属部门 / 汇报关系**：品牌部 / 公关部 / 市场部；汇报品牌总监、市场总监或公关负责人。
- **在组织中的层级位置**：品牌外部声誉和媒体关系岗位，负责媒体、KOL、活动、舆情和公关口径。
- **与设计 / 版师 / 样衣 / 打版岗位的边界**：不做服装设计、不打版；不等同于内容运营，PR 更关注第三方背书、媒体关系、舆情和品牌声誉。

## 2. 核心职能
1. 通过媒体、达人、明星和行业资源建立品牌可信度和外部声量。
2. 为新品、品牌事件、合作和活动设计公关传播节奏。
3. 维护媒体和 KOL 关系，筛选合作对象并管理稿件/brief/样衣借用。
4. 监测舆情和竞品公关动态，及时给出风险处理建议。
5. 将品牌信息包装成适合媒体和社交传播的新闻点与故事。

## 3. 典型工作节奏
- **日常**：维护媒体/达人沟通、跟进稿件、样衣寄送、合作进度和舆情监测。；给品牌和内容团队反馈媒体选题、KOL 反馈和传播机会。
- **周期性**：每周更新传播进度和声量；每月复盘媒体/KOL 效果和资源库。；每季配合新品发布、订货会、品牌活动和大促传播。
- **触发式**：新品发布、明星穿搭、负面舆情、质量投诉、联名官宣、媒体采访时触发专项 PR 动作。

## 4. 协作关系
- **上游**：品牌营销给传播主题；商品给新品卖点；内容给素材；法务给合规边界；高层给采访口径。
- **下游**：媒体、达人、内容、电商、客服、法务使用 PR 稿件、brief、问答和舆情建议。
- **横向协作最频繁的 3 个岗位**：品牌营销、内容/社媒、法务/合规。

## 6. 关键决策点

| 决策内容 | 决策依据 | 决策错误的代价 |
|---|---|---|
| 选择哪些媒体/KOL | 受众匹配、内容质量、历史效果、报价、风险记录 | 选错会浪费预算或引发负面。 |
| 公关口径怎么写 | 事实证据、品牌立场、法务边界、消费者情绪 | 口径不当会扩大舆情。 |
| 是否回应负面舆情 | 传播范围、真实性、平台热度、法律风险 | 沉默或过度回应都可能放大风险。 |
| 样衣借给谁、借多少 | 资源稀缺、曝光价值、归还风险、上市保密 | 管理不善会丢失样衣或提前泄露。 |
| 新闻点如何包装 | 品牌战略、媒体兴趣、产品卖点、社会话题 | 新闻点弱则传播无法被采用。 |

## 7. 专业知识与术语

### 行业黑话 / 缩略语
PR；Press Kit；通稿；媒体邀约；媒体露出；Earned Media；Owned Media；Paid Media；KOL；KOC；MCN；达人 brief；样衣外借；舆情；危机公关；声量；情感分析；SOV；新闻点；发言人；采访 Q&A；禁用词；官宣；软文；硬广；媒体价值；转载；背书。

### 常用 KPI 名词及计算口径
| KPI | 常用口径 |
|---|---|
| 媒体露出量 | 有效报道/提及数量 |
| 正负面情感比例 | 正面/中性/负面提及占比 |
| KOL 内容完成率 | 按期交付内容数/计划内容数 |
| 传播触达 | 媒体和社媒曝光估算 |
| 危机响应时效 | 舆情发现到首次响应时间 |

### 日常使用的软件 / 系统 / 模板
媒体资源 CRM、舆情监测工具、社媒后台、Excel 资源库、网盘 Press Kit、邮件、飞书审批。

### 必懂的行业基础知识
懂媒体运作、时尚传播节奏、KOL 合作、舆情分级、广告法/代言合规基础、品牌口径和样衣管理。

## 8. 痛点 / 低效环节
- 媒体和达人资源信息靠个人维护，交接困难。
- PR 效果难和销售直接归因，价值证明难。
- 危机时事实、法务、客服信息不同步。
- 样衣外借和归还跟踪繁琐。
- 通稿容易过度营销，媒体采用率低。

## 9. 典型用户问句
1. 帮我把这次新品发布写一版新闻稿。
2. 这个 KOL 报价 3 万，历史互动一般，要不要合作？
3. 负面小红书笔记说我们面料掉色，PR 该不该回应？
4. 帮我生成媒体采访 Q&A，避开成本和供应商敏感信息。
5. 把本月媒体露出按正负面和主题分类。
6. 这场发布会需要哪些 press kit 材料？
7. 给达人写一版羊绒大衣合作 brief。
8. 竞品请了明星街拍，我们可以用什么低成本 PR 动作跟进？
9. 危机声明第一段怎么写更稳妥？
10. 样衣外借登记表帮我补充风险字段。
