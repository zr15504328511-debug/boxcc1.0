## 1. 岗位标识
- **中文名 / 英文名 / 常见别名**：商品分析师 / 商品运营 / Merchandise Operations / Merchandise Analyst / 商品专员、货品运营、商品数据分析、调拨专员、品类运营。
- **所属部门 / 汇报关系**：商品中心 / 零售运营 / 电商商品部；通常汇报商品经理、商品运营经理或渠道商品负责人。
- **在组织中的层级位置**：执行与分析并重的经营岗位，连接企划、销售、仓配和财务；大型品牌会按渠道或品类分组。
- **与设计 / 版师 / 样衣 / 打版岗位的边界**：不定义服装设计语言，不打版、不看样衣工艺细节；重点是让已确定的货品在正确渠道、时间和折扣下销售。

## 2. 核心职能
1. 监控进销存和商品生命周期，及时发现缺货、积压、断码和折扣异常。
2. 把销售目标拆到商品、渠道、门店或平台，并跟踪达成差距。
3. 通过补货、调拨、退仓、组合销售和促销建议提升售罄与毛利。
4. 为订货、补单、清仓、活动货盘提供数据依据。
5. 维护商品基础资料、价格、库存和配发规则，保证前端卖货数据准确。

## 3. 典型工作节奏
- **日常**：看昨日销售、库存、断码、退货、活动表现，处理补货/调拨/退仓申请。；更新商品日报，提醒渠道或门店处理滞销、缺码和价格错误。
- **周期性**：每周输出畅滞销和库销风险；每月做品类、渠道、门店商品效率复盘。；大促前做货盘测算和库存锁定；季末做清仓和退仓计划。
- **触发式**：爆款缺货、库存过高、门店开闭店、渠道活动、价格调整、仓库盘点差异时触发商品动作。

## 4. 协作关系
- **上游**：企划给商品结构和目标；销售/电商/门店给销售反馈；仓储给库存状态；财务给毛利和成本口径。
- **下游**：给渠道、门店、电商、仓储、采购、供应链提供补调退、促销、补单和清仓建议。
- **横向协作最频繁的 3 个岗位**：零售门店运营、电商运营、仓储物流。

## 6. 关键决策点

| 决策内容 | 决策依据 | 决策错误的代价 |
|---|---|---|
| 哪些 SKU 今天优先补货 | 近7天销量、现存、在途、铺货等级、活动排期 | 补货不足会断货；过度补货会门店积压。 |
| 滞销款打几折、在哪些渠道清 | 库销比、毛利、品牌形象、渠道客群、季节窗口 | 折扣过早伤毛利和价格心智；过晚形成死库存。 |
| 爆款是否翻单 | 售罄速度、面辅料可得性、生产交期、剩余销售周期 | 翻单错过窗口或造成晚到货库存。 |
| 库存从哪个门店/仓调出 | 门店销售等级、库存深度、物流成本、区域差异 | 调错会牺牲高销门店机会。 |
| 活动货盘是否够卖 | 目标 GMV、转化率、库存、仓配能力、售后风险 | 货不够浪费流量，货过多造成活动后库存。 |

## 7. 专业知识与术语

### 行业黑话 / 缩略语
进销存：采购/销售/库存；动销率；售罄率；库销比；周转天数；安全库存；铺货率；补货；调拨；退仓；断码；断色；缺货率；库龄；吊牌价；成交价；折扣率；毛利率；在途库存；可售库存；冻结库存；预占库存；TOP 款；TAIL 款；尾货；清仓；翻单。

### 常用 KPI 名词及计算口径
| KPI | 常用口径 |
|---|---|
| 售罄率 | 累计销量/累计到货 |
| 动销率 | 有销售 SKU 数/可售 SKU 数 |
| 库存周转天数 | 平均库存金额/日均销售成本 |
| 缺货率 | 缺货 SKU 或缺货订单/应满足 SKU 或订单 |
| 折扣率 | 成交价/吊牌价 |

### 日常使用的软件 / 系统 / 模板
ERP、POS、OMS、WMS、BI、Excel 透视表、商品主数据表、钉钉/飞书审批、天猫生意参谋、抖店罗盘。

### 必懂的行业基础知识
必须懂款色码、生命周期、渠道铺货等级、折扣与毛利关系、补调退规则、库存占用和大促备货逻辑。

## 8. 痛点 / 低效环节
- 每天在多个系统导数，SKU 口径和时间口径不一致。
- 补调退审批链长，等批完销售窗口已过。
- 门店反馈零散，商品问题归因困难。
- 活动货盘经常临时变化，库存冻结和价格维护容易出错。
- 库存预警靠经验阈值，难兼顾季节、渠道和活动。

## 9. 典型用户问句
1. 帮我把这周销售数据按品类、渠道、折扣段拉一下。
2. 这 20 个款的库销比都超过 6，按清仓优先级排一下。
3. 上海 3 家店缺 M 码，哪里可以调货最不影响销售？
4. 双11活动货盘库存够支撑 300 万 GMV 吗？
5. 这款上市 10 天动销 18%，是滞销还是铺货不足？
6. 帮我写一封调拨异常催办邮件给区域经理。
7. 把昨日 TOP30 款按毛利贡献和售罄做四象限。
8. 这批冬装要不要提前退仓？给个依据。
9. 帮我生成新品上市第 14 天复盘 PPT 大纲。
10. 这款 SKU 库存深但退货率也高，要不要继续投活动？
