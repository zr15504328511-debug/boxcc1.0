## 1. 岗位标识
- **中文名 / 英文名 / 常见别名**：HR / 培训 / 组织发展 / Human Resources / Training / Organization Development / HRBP、招聘、薪酬绩效、零售培训、组织发展、企业大学。
- **所属部门 / 汇报关系**：人力资源部 / 培训部 / 组织发展部；汇报 HRD、人力资源经理或总经理；培训岗常服务零售、销售和客服。
- **在组织中的层级位置**：组织能力岗位，负责招聘、绩效、培训、人才梯队和组织机制；零售品牌中培训常是销售能力中台。
- **与设计 / 版师 / 样衣 / 打版岗位的边界**：不做服装设计和生产；培训可讲产品知识和搭配话术，但基于商品/设计提供的信息，不创造实物标准。

## 2. 核心职能
1. 为门店、销售、电商、供应链等岗位配置合适的人，并降低关键岗位流失。
2. 把品牌、商品、服务和管理要求转成培训课程和可考核的行为标准。
3. 设计绩效、激励和晋升机制，让组织目标与个人行为对齐。
4. 支持组织调整、人才盘点和管理者培养，提升团队执行力。
5. 用员工数据识别招聘、培训、离职、绩效和组织氛围问题。

## 3. 典型工作节奏
- **日常**：处理招聘、入转调离、员工问题、培训安排、绩效答疑。；跟进门店/部门用人缺口和培训反馈。
- **周期性**：每周招聘进度和培训执行；每月人力报表、离职分析、绩效跟进。；每季新品培训、人才盘点、绩效评估；每年组织架构和薪酬预算。
- **触发式**：新店开业、大促临时用工、离职潮、绩效争议、组织调整、新品上市时触发专项 HR/培训动作。

## 4. 协作关系
- **上游**：业务部门给岗位需求、人员反馈和能力缺口；商品/品牌给培训内容；财务给人力预算。
- **下游**：门店、销售、电商、客服、供应链、管理层使用招聘计划、培训材料、绩效方案和人才报告。
- **横向协作最频繁的 3 个岗位**：零售门店运营、品牌营销、财务预算。

## 6. 关键决策点

| 决策内容 | 决策依据 | 决策错误的代价 |
|---|---|---|
| 候选人是否录用 | 能力匹配、薪酬、稳定性、文化、用人紧急度 | 误招会带来培训成本和团队风险。 |
| 培训重点放哪里 | 业务指标、门店反馈、客诉、商品变化、考试结果 | 培训错位无法改善业绩。 |
| 绩效指标怎么设 | 公司目标、岗位影响、可控性、数据可得性 | 指标不合理导致行为扭曲或争议。 |
| 离职风险如何干预 | 离职率、访谈、主管反馈、薪酬市场、排班压力 | 干预不及时会造成关键岗位缺口。 |
| 组织是否需要调整 | 业务增长、流程瓶颈、人效、管理跨度、成本 | 调整过度伤士气，不调整影响执行。 |

## 7. 专业知识与术语

### 行业黑话 / 缩略语
HRBP；JD；ATS；HC：编制；入转调离；试用期；胜任力；绩效；OKR；KPI；薪酬带宽；提成；奖金池；人效；离职率；到岗率；培训覆盖率；考试通过率；人才盘点；九宫格；继任计划；组织诊断；敬业度；LMS；SOP；带教；门店梯队；新店筹备。

### 常用 KPI 名词及计算口径
| KPI | 常用口径 |
|---|---|
| 招聘到岗率 | 实际到岗人数/计划招聘人数 |
| 离职率 | 离职人数/平均人数 |
| 培训通过率 | 通过考试人数/参训人数 |
| 人效 | 销售额或产出/人数或人时 |
| 绩效达成率 | 达成目标人数/参与考核人数 |

### 日常使用的软件 / 系统 / 模板
ATS 招聘系统、HRIS、LMS、绩效系统、Excel 人力报表、钉钉/企业微信、问卷工具、知识库。

### 必懂的行业基础知识
懂零售门店和电商岗位能力模型、销售提成、排班、培训转化、劳动合规、组织架构和人才梯队。

## 8. 痛点 / 低效环节
- 门店招聘和离职高频，HR 被动补缺。
- 培训材料来自商品/品牌/客服多方，更新不及时。
- 培训效果难与销售指标关联。
- 绩效指标经常被质疑不可控或口径不清。
- 组织调整沟通敏感，需要兼顾效率和员工情绪。

5. 帮我设计门店店长绩效指标和权重。

7. 新品培训考试出 10 道选择题。
8. 组织架构调整公告怎么写更稳妥？
9. 帮我分析培训通过率和店效提升是否有关。
10. 导购连带率低，培训话术怎么改？
