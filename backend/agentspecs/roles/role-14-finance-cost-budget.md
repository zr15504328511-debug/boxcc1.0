## 1. 岗位标识
- **中文名 / 英文名 / 常见别名**：财务 / 成本会计 / 预算 / Finance / Cost Accounting / Budgeting / 服装会计、成本会计、财务 BP、预算分析、应收应付、结算。
- **所属部门 / 汇报关系**：财务部 / 经营管理部；汇报财务经理、CFO 或总经理；成本会计常对接供应链和商品。
- **在组织中的层级位置**：经营控制和核算岗位，负责收入成本费用、库存资金、预算、毛利和对账；中型公司常分核算、成本、预算和电商财务。
- **与设计 / 版师 / 样衣 / 打版岗位的边界**：不决定款式设计、不做销售执行；财务提供成本、预算、风险和合规约束，业务负责人拍板。

## 2. 核心职能
1. 把服装采购、生产、销售、库存和费用转成准确的财务账和经营利润。
2. 核算单品/订单/渠道成本与毛利，帮助业务理解价格、折扣和库存对利润的影响。
3. 编制预算和滚动预测，控制费用、现金流、应收应付和库存资金占用。
4. 对接电商平台、渠道客户、供应商和仓库，完成结算、对账、发票和付款。
5. 识别财务风险，如坏账、亏损促销、成本偏差、库存减值和税务合规问题。

## 3. 典型工作节奏
- **日常**：审核报销、付款、收入、平台账单、采购入库和库存变动。；回答业务关于成本、毛利、预算余额和费用审批的问题。
- **周期性**：每周跟进应收应付和现金；每月结账、成本核算、预算执行和经营分析。；每季/每年做预算、审计、库存跌价和经营复盘。
- **触发式**：大促、价格调整、供应商涨价、库存清仓、合同变更、税务检查时触发财务测算或审查。

## 4. 协作关系
- **上游**：商品、采购、供应链、电商、渠道提供订单、合同、价格、收入和库存；BI/IT 提供数据。
- **下游**：管理层、业务部门、法务和供应商使用财务报表、成本测算、预算意见和付款结算。
- **横向协作最频繁的 3 个岗位**：采购、商品运营、销售渠道运营。

## 6. 关键决策点

| 决策内容 | 决策依据 | 决策错误的代价 |
|---|---|---|
| 某活动是否亏损可做 | 成交价、成本、平台扣点、广告、物流、退货、库存目标 | 误判会造成销售看似增长但利润亏损。 |
| 库存是否计提跌价 | 库龄、季节性、预计售价、清仓计划、成本 | 不计提虚增利润，过度计提影响业绩。 |
| 是否批准费用或付款 | 预算、合同、发票、验收、现金流、合规 | 付款错误造成资金损失或供应商关系问题。 |
| 客户授信/账期是否放宽 | 历史回款、信用、订单利润、库存压力 | 坏账和现金流风险。 |
| 成本差异是否追责 | 采购价、单耗、损耗、生产异常、汇率税费 | 不追踪会侵蚀毛利，追错会伤害协作。 |

## 8. 痛点 / 低效环节
- 业务系统与财务系统口径不同，结账期大量对账。
- 电商平台扣费复杂，订单级利润难算。
- 商品成本在开发期不断变化，毛利预测滞后。
- 预算审批与项目执行脱节，超支后才发现。
- 库存减值和清仓计划需要业务判断，证据收集耗时。
